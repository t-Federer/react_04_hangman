# Hangman in React
Gioco in cui il giocatore deve indovinare la parola misteriosa.
Il giocatore deve indovinare quali lettere fanno parte della parola, ma ogni lettera sbagliata gli "consuma punti vita", qui rappresentati dai linguaggi di programmazione.
Il gioco finisce quando il giocatore ha fallito (tutti i linguaggi sono stati eliminati), oppure quando indovina la parola.
